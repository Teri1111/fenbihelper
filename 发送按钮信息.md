### 一、控件层次结构
从截图的布局树可以看出，目标“发送”按钮的完整层级如下（从根到叶子节点）：
```
android.view.ViewGroup (最外层容器)
└── FrameLayout
    └── android.view.ViewGroup
        └── FrameLayout
            └── LinearLayout
                └── FrameLayout
                    └── android.view.View
                        └── FrameLayout
                            └── ImageView (目标控件)
```
**层级深度**：`depth = 13`，说明该按钮嵌套在13层布局容器中。

### 二、按钮核心属性
目标控件是一个 `android.widget.ImageView`，同时作为“发送”按钮，关键属性如下：

| 属性名 | 属性值 | 含义 |
|--------|--------|------|
| `className` | `android.widget.ImageView` | 控件类型为图片视图 |
| `id` | `action_send` | 控件ID，用于代码中定位 |
| `fullId` | `com.larus.nova:id/action_send` | 完整资源ID，包含包名 |
| `idHex` | `0x7f0a031b` | 控件ID的十六进制表示 |
| `text`/`desc` | `发送` | 无障碍描述/内容描述，即按钮功能 |
| `bounds` | `(1091,1491,1233,1672)` | 屏幕坐标范围（左, 上, 右, 下） |
| `clickable` | `true` | 可点击 |
| `enabled` | `true` | 处于启用状态 |
| `focusable` | `true` | 可获取焦点 |
| `contextClickable` | `false` | 不可上下文点击 |
| `longClickable` | `false` | 不可长按 |
| `editable` | `false` | 不可编辑 |
| `scrollable` | `false` | 不可滚动 |
| `checked` | `false` | 未选中状态 |
| `selected` | `false` | 未选择状态 |
| `dismissable` | `false` | 不可关闭 |
| `drawingOrder` | `1` | 绘制顺序为1 |
| `indexInParent` | `0` | 在父容器中的索引为0 |
| `packageName` | `com.larus.nova` | 所属应用包名 |

### 三、补充说明
- 该控件本质是 **ImageView**，通过设置 `desc="发送"` 实现了按钮的无障碍功能与视觉提示，并非传统的 `Button` 或 `AppCompatButton`。
- 从 `bounds` 坐标可以看出，它位于屏幕右下角区域，符合聊天界面“发送按钮”的典型位置。
